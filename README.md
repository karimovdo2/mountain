# Горный год — Streamlit-приложение

Приложение рисует таймлайн походов в виде PNG-гор.

## Структура

- `app.py` — основной код Streamlit-приложения.
- `requirements.txt` — зависимости.
- Папка `maunt/` — сюда нужно положить 7 PNG-файлов с горами.
- Файл `походы.csv` — данные из Garmin со столбцами как минимум:
  - `Дата`
  - `Максимальная высота`

Дополнительно, для сводной статистики, можно иметь столбцы:
  - `Расстояние` (в километрах)
  - `Подъем` (набор высоты, в метрах)

## Запуск

1. Создайте и активируйте виртуальное окружение (по желанию).
2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Положите `походы.csv` и папку `maunt` с PNG-изображениями в ту же директорию.
4. Запустите приложение:

   ```bash
   streamlit run app.py
   ```

5. Откройте ссылку из терминала (обычно `http://localhost:8501`) в браузере.

В боковой панели можно настраивать толщину гор и параметры разнесения подписей высоты.
